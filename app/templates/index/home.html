ABOUT THIS PROJECT

This is a DBMS project which is based on the Flask framework. The flask framework is a built-in development server and a fast debugger. It has an integrated support for unit testing and a Jinja2 templating. It supports for secure cookies and is lightweight. It's modular design helps for a flexible framework. Hence, considering these features of the flask, it's been used in this DBMS project. In this project, the user can access the particular artist, genre, language and the desired type of the music he wishes to.
